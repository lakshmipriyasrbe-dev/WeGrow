## Installation

1. Clone the repository

```bash
git clone https://github.com/lakshmipriyasrbe-dev/WeGrow.git
```

2. Move the project to your web server directory

```text
xampp/htdocs/
```

3. Import the database

```text
database.sql
```

4. Update database credentials

```php
$db_host = "localhost";
$db_user = "root";
$db_pass = "";
$db_name = "wegrow";
```

5. Start Apache and MySQL

6. Open in browser

```text
http://localhost/WeGrow
```
